The FicLibrary API

This API provides information about Harry Potter fanfiction stories I have read from Archive of Our Own (Archive for transformative fanworks). My main purpose is to greate archive where I can check what fics I have read and if they were good.


The routes are following: Getting data of all fics: https://ficlibrary.herokuapp.com/api/getall Getting data by a specifid ID number: https://ficlibrary.herokuapp.com/api/"ID" Getting data by a specific fics name: https://ficlibrary.herokuapp.com/api/name/"NAME"
